# Zed Skill Name Rules

## Editable fields

- Lowercase ASCII letters and numbers.
- Hyphens as separators only.
- Prefix CE skills with `ce-`.
- Limit skill name length to 64 characters.

## Examples

Valid: `ce-code-review`, `ce-plan`, `ce-brainstorm`

Invalid: `CodeReview`, `ce_code_review`, `ce-code-review-extremely-long-name-that-exceeds-the-maximum-length-because-this-is-too-long`
