# Cursor plugin setup

Install Compound Engineering slash commands for **any** Cursor workspace.

**Option A (script only):** from the monorepo root, run `npm run install:cursor`, then quit Cursor fully (**Cmd+Q**) and reopen.

**Option B (paste to agent):** attach or paste `docs/cursor-agent-setup.md` and ask the agent to complete setup.

Do not hand-roll symlinks or `installed.json` edits when the script is available.
